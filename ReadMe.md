## 개발 방법

Agile 개발 방법론의 Scrum방식으로 짧은 주기를 가진 sprint를 진행하며 개발합니다.

## 사용하는 기술

**형상관리 도구** : Git

**프로젝트 관리 도구** : Jira

**커뮤니케이션 도구** : slack

**프레임워크** : React
